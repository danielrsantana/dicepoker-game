# react-fizzbuzz-game
ReactJS + Typescript &amp; Jest FizzBuzz Game

## description
Game Rules: Numbers dived by 3: Fizz; Numbers Divided by 5: Buzz; Numbers divided by both, FizzBuzz
(and for the rest? Display the number)

## setup development:
```
  npm install
  npm start
```

## setup production:
```
  npm build
```

## Unit Testing:
```
  npm run test
```
